# omnibot-v2

This is a bespoke discord bot written in python, with discord.py library, for OmniGamers Community.

Before you run the main script file, create an .env file that contains the following variables:
```  
  BOTNAME=
  TOKEN=
  DBNAME=
  DBHOST=
  DBPORT=
  DBUSER=
  DBPASSWD=
  GUILD_ID=
```
